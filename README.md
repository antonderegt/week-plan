# WeekPlan

A meal planning app for scheduling weekly dinners, managing recipes, and generating shopping lists.
See [WeekPlan MCP](https://github.com/antonderegt/week-plan-mcp) on how to use an MCP server to enter the recipes for you.
See [WeekPlan Watch](https://github.com/antonderegt/week-plan-watch) on how to setup an Apple Watch companion app.

## Getting started

```bash
npm install
npm run dev
```

## Importing example recipes

The `examples/` directory contains a starter set of recipes you can import to get going quickly.

1. Open the app and go to **Settings** (bottom tab on mobile, top nav on desktop).
2. Scroll to the **Data** section and click **Import data**.
3. Select `examples/starter-recipes.json` from this repository.
4. Confirm the prompt — this replaces all existing data.

The starter pack includes six recipes: lemon herb chicken, beef bolognese, red lentil & spinach curry, teriyaki salmon, chicken burritos, and creamy chicken & mushroom pie.

You can also export your own data at any time from Settings → Export data, and re-import it later as a backup.
